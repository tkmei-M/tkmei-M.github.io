## キズナランク計算機

スマートフォンリズムゲームのスクリーンショットから、目標ランクまでの必要プレイ回数や時間を計算するツールです。

### 使い方
1. キズナランク画面のスクリーンショットをアップロードします。
2. 目標ランク、スコアランク、ライブボーナス消費量、楽曲を選択します。
3. 「認識開始」をクリックすると、OCR（光学文字認識）によって現在の状況が読み取られ、計算結果が表示されます。

### 技術スタック
- HTML / CSS / JavaScript
- [Tesseract.js](https://github.com/naptha/tesseract.js) (OCR Engine)
